# FMCG Promotion ROI Intelligence Platform

This repository is the starting point for an end-to-end ML portfolio project that predicts promotion ROI, explains predictions with SHAP, and serves them through a real-time API on Kubernetes.

## Current status

- Repository scaffold created
- Docs folder initialized as learning stubs
- Training, serving, ranker, pipeline, and Kubernetes directories are in place

## Planned phases

1. Data layer and BigQuery feature engineering
2. Docker training/serving containerization
3. Vertex AI training and model registry
4. GKE deployment and live API
5. Autoscaling, rolling updates, and monitoring
6. Full retraining pipeline with Vertex AI Pipelines

## Next step

Start with Phase 1: confirm GCP prerequisites and prepare the Rossmann dataset pipeline.
